## 版本

FSRS-6，21 个参数 w[0] 到 w[20]。

## 辅助常量

```
factor = 0.9^(-1/w20) - 1     （保证 t=S 时 R=90%）
decay  = -w20
```

## 一、首次见面（一道题第一次被做）

### 1.1 初始稳定性 S0

```
S0(Again) = w0 = 0.5707
S0(Hard)  = w1 = 1.9849
S0(Good)  = w2 = 4.3528
S0(Easy)  = w3 = 10.0024
```

答得越好，初始 S 越高。

### 1.2 初始难度 D0

```
D0(G) = w4 - e^(w5 × (G-1)) + 1

D0(Again)  = w4 - 1 + 1 = w4 = 6.4133     （最难）
D0(Hard)   = w4 - e^w5 + 1 ≈ 4.82
D0(Good)   = w4 - e^(2×w5) + 1 ≈ 3.27
D0(Easy)   = w4 - e^(3×w5) + 1 ≈ 1.67     （最简单）
```

答得越好，初始 D 越低。结果 clamp 在 [1, 10]。

## 二、遗忘曲线（算 R）

### 2.1 正向：已知距上次过了 t 天，记忆留存率？

```
R(t, S) = (1 + factor × t / S)^(-w20)
```

验证边界：t = S 时，R = (1 + (0.9^(-1/w20)-1) × 1)^(-w20) = 0.9 ✓

### 2.2 反向：给定目标保留率 r，下次隔几天？

```
I(r, S) = S / factor × (r^(1/(-w20)) - 1)
```

实际使用时还加了一个 fuzz（±5% 随机抖动），防止同一批题总在同一天出现。

---

## 三、每次复习后的更新

### 3.1 更新难度 D

```
ΔD(G) = -w6 × (G - 3)

G=Again  → ΔD = +2×w6    （变难）
G=Hard   → ΔD = +w6       （略难）
G=Good   → ΔD = 0         （不变）
G=Easy   → ΔD = -w6       （变简单）
```

然后经过两个步骤应用到旧 D：

```
步骤1：线性缩放
    新D1 = D + ΔD × (10 - D) / 9

步骤2：均值回归（拉向 D0(Easy)，防止难度爆炸）
    新D2 = w7 × D0(Easy) + (1 - w7) × 新D1

最终 clamp 在 [1, 10]
```

w6=3.0194 控制评分冲击力度。w7=0.001 控制回归力度——几乎不做回归，难度由评分说了算。D0(Easy) ≈ 1.67，拉回方向是"偏简单"。

---

### 3.2 更新稳定性 S —— 同日反复复习

```
S'(S, G) = S × e^(w17 × (G - 3 + w18)) × S^(-w19)

SInc（增长率） = e^(w17 × (G-3+w18)) × S^(-w19)

约束：当 G ≥ 3 时，必须 SInc ≥ 1
```

- w17: 评分对同日复习的冲击力
- w18: 让 Good 评分的同日复习也有小幅增长
- w19: S 越大 → 同日复习效果越衰减

当 SInc = 1 时，S 收敛不涨了——达到同日复习的天花板。

---

### 3.3 更新稳定性 S —— 答对（跨天复习）

这是 FSRS 最核心的公式：

```
S'_r(D, S, R, G) = S × (
      e^w8
    × (11 - D)
    × S^(-w9)
    × (e^(w10 × (1 - R)) - 1)
    × HARD    （G=Hard 时 = w15，否则 = 1）
    × EASY    （G=Easy 时 = w16，否则 = 1）
    + 1
)
```

逐项直觉：

| 因子 | 来源参数 | 含义 |
|------|---------|------|
| e^w8 | w8=1.8722 | 基础增长倍数 |
| (11-D) | — | D 大（难题）→ 涨得少 |
| S^(-w9) | w9=0.1666 | S 大（已熟练）→ 涨得少 |
| e^(w10×(1-R))-1 | w10=0.796 | R 小（隔得久但没忘）→ 大涨（间隔效应） |
| w15 | w15=0.6014 | 按 Hard 时打六折 |
| w16 | w16=0.8729 | 按 Easy 时加乘 |
| +1 | — | 保证 SInc ≥ 1，稳定性不降 |

---

### 3.4 更新稳定性 S —— 答错（遗忘后稳定性）

```
S'_f(D, S, R) = w11
    × D^(-w12)
    × ((S + 1)^w13 - 1)
    × e^(w14 × (1 - R))
```

逐项直觉：

| 因子 | 来源参数 | 含义 |
|------|---------|------|
| w11 | w11=1.4835 | 基础遗忘后残留 |
| D^(-w12) | w12=0.0614 | D 大（难题）→ 遗忘后残留更少 |
| (S+1)^w13-1 | w13=0.2629 | 原来的 S 大 → 打折后残留也越多。不会归零 |
| e^(w14×(1-R)) | w14=1.6483 | 隔了很久才回来 → 给一点补偿 |

结果还要和 `S / e^(w17×w18)` 取最小值——遗忘后稳定性不得超过原 S 的某个比例。

w12=0.0614 很小，说明 FSRS 对 D 在遗忘公式里的依赖很轻。
w14=1.6483 较大，说明"隔了很久回来"会有较多补偿。

## 四、完整流程回顾

```
用户答完一道题，给出评分 G

如果这道题是 NEW：
    D = D0(G)          （公式 1.2）
    S = S0(G)          （公式 1.1）
    间隔 = I(r, S)     （公式 2.2）

如果 G == Again（答错）：
    R = R(t, S)        （公式 2.1）
    D = nextD(D, G)    （公式 3.1）
    S = S'_f(D, S, R)  （公式 3.4）
    间隔 = I(r, S)

如果是同日复习：
    D = nextD(D, G)
    S = S'_sameDay(S, G)  （公式 3.2）
    间隔 = I(r, S)

否则（答对，跨天）：
    R = R(t, S)
    D = nextD(D, G)
    S = S'_r(D, S, R, G)  （公式 3.3）
    间隔 = I(r, S)

存回数据库：D, S, due = now + 间隔
```

## 五、21 个参数的职责速查

| 参数 | 用途 |
|------|------|
| w[0..3] | 首次见到题的初始 S（按 G=1..4） |
| w[4..5] | 初始难度 D0 公式 |
| w[6] | 评分对难度的冲击力度 |
| w[7] | 难度均值回归力度（越小越听评分的） |
| w[8] | 基础稳定性增长倍数 |
| w[9] | S 越大涨越慢的衰减指数 |
| w[10] | 间隔效应强度（R 小则大涨） |
| w[11..14] | 遗忘后稳定性公式的四个参数 |
| w[15] | Hard 评分的惩罚系数 |
| w[16] | Easy 评分的奖励系数 |
| w[17..19] | 同日反复复习公式的三个参数 |
| w[20] | 遗忘曲线衰减形状（越大忘越快） |
